
# AVAILABLE TOOLS
1. **Write File**:
```write path/to/file
Content here...
```
2. **Read File**:
```read path/to/file```
3. **List Directory**:
```list path/to/dir```
4. **Run Command**:
```bash
cmd args
```

# RULES
1. Use `write` blocks for ALL file creation/edits. DO NOT use `cat` or `echo` redirection.
2. Wait for the result before proceeding.
3. CRITICAL: Do NOT put commentary inside the code block.
